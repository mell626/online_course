System description

A demo website for uploading and viewing of new videos. This platform is inspired by freecodecamp.org design. The developer of this project is fascinated the way freecodecamp.org provide free educational contents using the web. The system is mainly used to upload videos by the administrator (preferrably educational videos) and can be played by the users at the other end of the system. The files that can be uploaded are jpeg, png, wav and mp4 files. As the end user (student), uploading videos is prohibited. The system's current version is at its basic framework and is subject for improvements.
